# Sweviceroy MTG

A simplified MTG-inspired web card game built in HTML, CSS and JavaScript, for our first labbassignment in the webbdeveloptment course at
ITHS Gothenburg.

## Features (MVP)
- Login system with localStorage
- Fake money and betting system
- 40-card decks
- Creatures + lands only
- Turn phases (draw, untap, main, combat, etc.)
- Combat system (attack & block)
- Win by life = 0 or depleted deck
